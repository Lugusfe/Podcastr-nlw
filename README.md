# Podcastr
Aplicação para navegação e reprodução de episódios de podcasts desenvolvida utilizando ReactJS com NextJs.
> Em desenvolvimento

## Features
- Listagem de Podcasts;
- Descrição de episódios;
- Reprodução de Podcasts;
  
##  Tecnologias Utilizadas

-  Typescript
-  ReactJS
-  NextJS
-  Sass
-  date-fns
-  axios
-  rc-slider
